# cyphercon2026

# Controls:
There are two switches (upper S1 and lower S2) and one button (B).

### Mode 0, Idle
- B: Serial TX
- S1 Press: Go to mode 1 (Set Alias)
- S1 Up: Go to mode 2 (Inbox)
- S2 Press/Up/Down: Go to mode 3 (Compose I)

### Mode 1, Set Alias
- B: Save, return to mode 0 (Idle)
- S1 Press: Cancel, return to mode 0 (Idle)
- S1 Up: Move cursor left
- S1 Down: Shift/Caps
- S2 Press: Move cursor right
- S2 Up/Down: Change character

### Mode 2, Inbox
- S1 Press: Cancel, return to mode 0 (Idle)
- S2 Press: Mark read

### Mode 3, Compose I
- S1 Press: Cancel, return to mode 0 (Idle)
- S2 Press: Go to mode 2 (Compose II)
- S2 Up/Down: Prev/Next contact

### Mode 4, Compose II
- B: Send, return to mode 0 (Idle)
- S1 Press: Cancel, return to mode 0 (Idle)
- S1 Up: Cursor left
- S1 Down: Shift/Caps
- S2 Press: Cursor right
- S2 Up/Down: Change character
